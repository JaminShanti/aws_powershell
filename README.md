# aws_powershell

## AWS Powershell Scripts

Powershell is alive and well in the 2 most popular Public Cloud Platforms in the world, AWS and Azure. Open Source solutions come last to windows so having a solid understanding of powershell allows for non stop development on the worlds most popular operating system.

### Scripts

*   **DR_create_VPC.ps1**: Creates the VPC for DR.
*   **DR_recover_mongoDBNode.ps1**: Recovers a MongoDB Node for DR.
*   **CreateAMIsForAllServers.ps1**: Creates AMIs for all servers.
*   **ExportSecurityGroups_v2.ps1**: Exports Security Group descriptions and IP permissions to JSON and text files.
*   **aws_ec2_register_route53.ps1**: Registers or updates Route53 DNS records (A/PTR) for EC2 instances.
